# Forms2NG Web

Angular 18 frontend for Forms2NG.

## Structure

- `app-legacy/`: preserved AngularJS/Bower application
- `src/`: Angular 18 source
- `public/`: static assets copied into the Angular build

## Commands

```bash
npm install
npm start
npm run build:prod
npm test
```

## API configuration

- Development: `http://localhost:8080/forms2ng`
- Production: `/forms2ng`
